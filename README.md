# Flask + Android Tilt Monitoring System

이 프로젝트는 Flask와 Android를 연동하여 실시간으로 센서 데이터를 처리하고, 이를 시각적으로 모니터링하는 시스템을 구현한 것입니다.

## 주요 기능
- Flask 서버: 실시간 센서 데이터 수집 및 처리
- Android 앱: 센서 데이터를 서버로 전송하고, 결과를 받아서 화면에 표시
- 실시간 그래프: 센서 데이터를 시각화하여 실시간으로 모니터링

## 시스템 구성
- **Flask Server**: 센서 데이터를 받아서 처리하고 결과를 제공
- **Android App**: 센서 데이터를 서버에 전송하고 결과를 표시
- **MPU6050 센서**: 가속도 및 기울기 데이터를 측정하여 서버로 전송
